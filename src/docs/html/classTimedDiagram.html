void TimedDiagram::segmentizeStochasticRegion ( Marking *  marking,
StochasticEvent *  eventLine,
double  timeBias 
)
Parameters
markingCurrent marking of the system.
eventLineThis is the equation of the lower boundary of region we wish to segmentize. \ Have in mind that this is a segment so it has the validity interval in itself.

Each (potential) region to be segmentized at first consists of starting and ending for 's' (like the previous intervals), and an event line, which specifies the equation for (the time after which the event has occurred) depending on . This function recursively segmentize such regions, by finding linear equations of all the possible next events. Note 1 : It is assumed that this function is called only for the upper part of st-diagram (when the general transition is fired.) Note 2 : Remember that initial marking for this function to start should be such that general transition is fired.
